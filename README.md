# Sales Dashboard Project

## Objective

To create an interactive dashboard using Power BI to analyze sales performance.

## Tools Used

* Power BI
* Excel Dataset

## KPIs

* Total Sales
* Total Profit
* Total Orders
* Profit Margin

## Features

* Interactive filters (Region, Category, Date)
* Time-series analysis
* Category and region insights

## Insights

* West region has highest sales
* Technology category gives more profit
* Sales trend shows growth over time

## Conclusion
“This Power BI dashboard was created as part of a data analytics project to visualize and analyze sales performance using interactive and dynamic visualizations.”

This dashboard helps businesses make data-driven decisions by visualizing key metrics effectively.
